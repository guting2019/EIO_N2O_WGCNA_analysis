# Bioinformatic and statistical workflow for epipelagic N2O cycling analyses

This repository contains the R workflow used to process 16S rRNA amplicon data, perform compositional data transformation, construct weighted correlation networks, relate microbial modules to environmental and N2O production variables, and generate network related outputs for the manuscript.

## Repository structure

```text
.
├── README.md
├── scripts
│   ├── 01_prepare_ASV_CLR.R
│   ├── 02_WGCNA_network_construction.R
│   ├── 03_module_trait_relationships.R
│   └── 04_ASV_importance_plots.R
├── data
│   ├── BacteriaASV.csv
│   ├── BacteriaASVtaxonomy.csv
│   └── N2OMetadata.csv
└── outputs
    ├── select.asv.clean.csv
    ├── EIO_asv_comb.csv
    ├── WGCNA_dataInput.RData
    ├── WGCNA-networkConstruction-auto.RData
    ├── MEs.csv
    ├── ModuleLabels_Colors.csv
    ├── TaxaTraitRelationships.csv
    └── figures
```

The `data` directory contains input files. The `scripts` directory contains the R scripts used for data processing and analysis. The `outputs` directory contains intermediate tables, final network tables, and figures generated by the workflow.

## Input files

### `BacteriaASV.csv`

ASV count table for bacterial 16S rRNA amplicon data.

Expected format:

```text
ASV_ID,Sample_1,Sample_2,Sample_3,...
ASV_1,10,0,32,...
ASV_2,0,5,18,...
```

Rows represent ASVs and columns represent samples. The first column should contain ASV identifiers.

### `BacteriaASVtaxonomy.csv`

Taxonomic annotation table for bacterial ASVs.

Expected format:

```text
ASV_ID,Kingdom,Phylum,Class,Order,Family,Genus,Tax
ASV_1,Bacteria,Proteobacteria,...
```

The first column should contain ASV identifiers that match those in `BacteriaASV.csv`.

### `N2OMetadata.csv`

Environmental and biogeochemical metadata table.

Expected format:

```text
Sample_ID,O2Mol,N2OMol,delta_N2O,NO2,NH4_Prod,NO3_Prod,NO2_Prod,Urea_Prod
Sample_1,...
```

The `Sample_ID` column should match the sample names in the ASV table.

Required variables include:

```text
O2Mol       dissolved oxygen concentration
N2OMol      dissolved N2O concentration
delta_N2O   excess N2O
NO2         ambient nitrite concentration
NH4_Prod    NH4 supported potential N2O production rate
NO3_Prod    NO3 supported potential N2O production rate
NO2_Prod    NO2 supported potential N2O production rate
Urea_Prod   urea supported potential N2O production rate
```

## R package requirements

The workflow was developed in R. The following packages are required:

```r
install.packages(c(
  "dplyr",
  "ggplot2",
  "openxlsx",
  "tibble",
  "grid",
  "WGCNA"
))

install.packages("BiocManager")
BiocManager::install(c(
  "phyloseq",
  "DESeq2"
))

install.packages(c(
  "zCompositions",
  "compositions"
))
```

Depending on the local R environment, `WGCNA` may require additional dependencies. Users should follow the installation instructions printed by R if any dependency is missing.


## Reproducibility notes

1. Sample names in the metadata table must match sample names in the ASV table.
2. ASV identifiers in the taxonomy table must match ASV identifiers in the ASV table.
3. All paths in the scripts assume the repository root as the working directory.
4. Random seed should be set before any stochastic step if additional random operations are added.
5. The WGCNA output may vary if filtering thresholds, zero replacement, transformation, or module merging parameters are changed.
